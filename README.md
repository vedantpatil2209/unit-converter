Unit Converter

This is a simple unit converter website I built using HTML, CSS, and JavaScript.
The idea was to create something useful while practicing core JavaScript concepts like DOM manipulation and event handling.

What it does

You can use this app to:

Convert between units of length, weight, and temperature
Select different categories and see units update automatically
Get quick results with a clean and simple interface
How to use
Enter a value (for example: 10)
Choose a category (Length, Weight, or Temperature)
Select the units you want to convert
Click Convert and see the result
